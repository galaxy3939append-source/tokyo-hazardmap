# 🗺️ 東京ハザードマップ

国土交通省の公式タイルデータをリアルタイム取得。**ズレなし・高精度**の東京都ハザードマップビューアです。

🌍 **ライブサイト:** https://galaxy3939append-source.github.io/tokyo-hazardmap/

---

## ✨ 特徴

- **ズレ一切なし** — 国交省タイルをリアルタイム配信
- **12種ハザード対応** — 洪水・内水・土砂・津波・高潮から選択
- **住所検索** — 国土地理院APIで即座に位置特定
- **複数レイヤー同時表示** — 複数の災害種を重ね合わせ確認
- **3種ベースマップ** — OSM / 地理院淡色 / 航空写真
- **軽量・オフライン対応** — CDN+キャッシュで高速

## 📖 使い方

### 1️⃣ ハザード種別を選ぶ

左パネルの **「ハザードレイヤー切替」** でチェック。複数同時にOK。

**洪水関連**
- 洪水浸水想定（想定最大規模）★デフォルト
- 洪水浸水想定（計画規模）
- 洪水浸水想定（国管理河川）
- 洪水浸水想定（東京都管理河川）
- 内水（雨水出水）浸水想定
- 浸水継続時間
- 家屋倒壊等氾濫想定

**土砂災害**
- 土砂災害警戒区域（土石流）
- 土砂災害警戒区域（急傾斜地）
- 土砂災害警戒区域（地すべり）

**その他**
- 津波浸水想定（東京都）
- 高潮浸水想定区域（東京都）

### 2️⃣ 透明度調整

**「透明度」** スライダーで0〜100%調整。

### 3️⃣ 住所検索

例: `調布市国領町` → マーカー表示

### 4️⃣ クイック移動

5市区町村＆東京都全域へジャンプ。

## 🎨 凡例

### 洪水浸水深
| 色 | 深さ | 説明 |
|---|---|---|
| 🔴 濃赤 | 5.0〜10.0m | 3階建て軒下 |
| 🟠 赤 | 3.0〜5.0m | 2階軒下 |
| 🟡 黄 | 0.5〜3.0m | 1階〜天井 |
| 🟨 薄黄 | 0.3〜0.5m | 腰まで |

### 土砂災害
| 色 | 種別 |
|---|---|
| 🔴 濃赤 | 特別警戒区域（レッドゾーン） |
| 🟠 茶色 | 警戒区域（イエローゾーン） |
